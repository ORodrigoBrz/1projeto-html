< H1 > até < H6 >

(Voltar)

As tags < h1 > até < h6 > são usadas para definir cabeçalhos (títulos) em uma página HTML, sendo o < h1 > o mais importante e maior, e o < h6 > o menos importante e menor. Elas ajudam a estruturar o conteúdo e melhorar a leitura.

Por exemplo:

Título Principal da Página

Introdução ao conteúdo da página.

Seção Principal

Detalhes sobre a seção principal.

Subseção

Mais informações sobre um tópico específico dentro da seção.

Subtópico da Subseção

Informações adicionais sobre o subtópico.

Detalhe Específico

Conteúdo que aprofunda um detalhe específico.

Nota de Rodapé ou Informações Menores

Pequenos detalhes ou observações finais.

< p >

(Voltar)

A tag < p > é usada para definir um parágrafo de texto. Qualquer conteúdo textual que você queira agrupar em um parágrafo deve estar dentro desta tag.

< mark >

(Voltar)

A tag < mark > representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

< Small >

(Voltar)

O elemento < small > em HTML representa comentários laterais e texto pequeno, como direitos autorais e texto legal, independentemente de sua apresentação estilizada.

Por padrão, ele renderiza o texto dentro dele um tamanho de fonte menor, como de pequeno para muito pequeno.

Por exemplo:

O preço do produto é R$ 99,99. (Nesse exemplo, o texto "R$ 99,99" ficará visualmente menor que o restante da frase. Isso é feito para indicar que é uma informação adicional ou que tem um peso semântico menor.)

< i >

(Voltar)

A tag < i > no HTML é utilizada para aplicar o estilo itálico ao texto, geralmente indicando que o texto tem um significado especial ou uma ênfase, embora visualmente ele apareça inclinado para a direita.

Por exemplo:

Eu gosto muito de futebol e basquete. (Obs: Isso fará com que as palavras "futebol" e "basquete" apareçam em itálico na página.)

< u >

(Voltar)

A tag < u > no HTML é usada para aplicar o estilo de sublinhado ao texto. Ela marca um trecho de texto que será visualmente sublinhado no navegador. Historicamente, o uso dessa tag era mais comum para indicar ênfase ou destaque, mas, com o tempo, a sua semântica foi repensada.

Por exemplo:

Este é um texto sublinhado. (Isso fará com que a palavra "texto sublinhado" apareça com uma linha abaixo dela).

< strong >

(Voltar)

A tag < strong > em HTML é usada para indicar ênfase forte em um texto, o que implica que o conteúdo dentro dessa tag tem uma importância especial dentro do contexto.

Visualmente, a tag < strong > normalmente aplica o estilo negrito ao texto, mas seu principal objetivo é semântico, ou seja, ela informa aos navegadores, leitores de tela e outros dispositivos de assistências que o texto marcado é significativo ou relevante.

Por exemplo:

Este é um texto com uma informação importante para ser destacada. (Isso fará com que a frase "informação importante" seja exibida em negrito, mas a ênfase semântica que a tag carrega é o que realmente importa.)

< ol > até < ul >

(Voltar)

A tag < ul > é usada para criar listas não ordenadas. Em uma lista não ordenada, a ordem dos itens não importa, ou seja, os itens podem ser apresentados em qualquer ordem.

Por exemplo:

  • Maçã
  • Banana
  • Laranja

(Aqui, o HTML cria uma lista com 3 itens, mas não há nenhuma ordem específica, apenas uma coleção de itens.)

A tag < ol > é usada para criar listas ordenadas. Em uma lista ordenada, a ordem dos itens é importante.

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

(Aqui, a lista é numerada automaticamente. A ordem dos itens é importante, indicando que eles devem ser seguidos sequencialmente).

< li >

(Voltar)

A tag < li > (List Item) no HTML é usada para definir um item dentro de uma lista. Ela é sempre utilizada dentro de uma lista criada pelas tags < ul > (lista não ordenada) ou < ol > (lista ordenada), ou em outros elementos de lista, como < menu > ou < nav >. A tag < li > representa um item individual da lista e é o principal elemento para estruturar as listas em HTML.

< a >

(Voltar)

A tag < a > (âncora) em HTML é utilizada para criar links (ou hiperligações) entre diferentes páginas ou recursos. O nome "âncora" vem da ideia de que ela ancora um ponto de navegação dentro de um documento, permitindo que o usuário seja redirecionado para outras páginas, seções da mesma página ou recursos externos

Por exemplo:

Visite o Google

(O texto "Visite o Google" será exibido como um link, e ao ser clicado, o usuário será redirecionado para a página do Google em uma nova aba do navegador, pois também foi utilizado o target="_blank", que será explicado a seguir).

Atributos importantes da tag < a >:

  • href: Especifica o destino do link. Pode ser um endereço de URL absoluto ou relativo.
  • target: Especifica onde abrir o destino do link. Pode ter os seguintes valores:
    • _self: Abre o link na mesma aba ou janela (padrão).
    • _blank: Abre o link em uma nova aba ou janela.
    • _parent: Abre o link no quadro pai (usado para páginas com frames).
    • _top: Abre o link na janela inteira (usado com frames).

< hr >

(Voltar)

A tag < hr > em HTML é usada para inserir uma linha horizontal que serve para separar conteúdos dentro da página, como uma divisão temática entre seções. Essa tag é autossuficiente, ou seja, não requer uma tag de fechamento.

Por Exemplo:

< p >Parágrafo 1< / p >

< hr >

< p >Parágrafo 2< / p >

(Ele irá formar a seguinte linha entre os parágrafos):

Parágrafo 1

Parágrafo 2

< sub > e < sup >

(Voltar)

A tag < sub > exibe o texto em uma posição ligeiramente abaixo da linha normal de texto, o que é comum para expressar fórmulas químicas ou índices.

Por exemplo:

Fórmula química da água: H2O

Já a tag < sup > exibe o texto em uma posição ligeiramente acima da linha normal de texto. Isso é comum para mostrar expoentes matemáticos ou abreviaturas de ordens (como números ordinais).

Por exemplo:

102 = 100

< blockquote >

(Voltar)

A tag < blockquote > em HTML é usada para indicar uma citação em bloco, geralmente uma citação mais longa e destacada do texto principal. Essa tag costuma aplicar uma indentação ao conteúdo, ajudando a separá-lo visualmente, o que é ideal para citações de parágrafos inteiros ou trechos de outros autores.

Por exemplo:

"O único limite para a nossa realização de amanhã são as nossas dúvidas de hoje." – Franklin D. Roosevelt

Atributos Comuns:

  • cite: Usado para especificar a URL da fonte da citação. Esse atributo não exibe um link diretamente, mas fornece uma referência que pode ser lida por navegadores e leitores de tela.

Por exemplo:

< blockquote cite="https://exemplo.com/citacao" >

"O conhecimento é poder." – Francis Bacon

< / blockquote >

< font >

(Voltar)

A tag < font > em HTML foi usada nas primeiras versões do HTML para definir o estilo de fonte, como cor, tipo e tamanho de texto. No entanto, a < font > foi descontinuada no HTML4 e removida do HTML5,

pois o controle de estilo foi transferido para o CSS, que oferece mais flexibilidade e organização.

Atributos e exemplo:

  • Color: Define a cor do texto. EX: < font color="blue" >Texto azul< /font>
  • Face: Define o tipo de fonte (font-family) do texto. EX: < font face="Arial">Texto em Arial< /font >
  • Size: Define o tamanho do texto em uma escala de 1 a 7. EX: < font size="4">Texto de tamanho 4< /font>

< del >

(Voltar)

A tag < del > em HTML é usada para marcar texto que foi deletado ou removido. O navegador geralmente exibe o texto com um traço horizontal sobre ele, indicando que algo foi "riscado" ou descontinuado.

Essa tag é útil para indicar alterações, revisões ou versões anteriores do conteúdo.

Por Exemplo:

Este produto não está mais disponível agora está em estoque novamente.

Código utilizado: < del >não está mais disponível< /del >

< abbr >

(Voltar)

A tag < abbr > em HTML é usada para marcar abreviações ou siglas, fornecendo um significado completo ou uma expansão para o texto quando o usuário passa o mouse sobre ele.

Essa tag melhora a acessibilidade ao descrever termos que podem não ser familiares para todos os leitores e pode ser lida por leitores de tela.

Por exemplo:

O HTML é a linguagem padrão para criar páginas web.

(Ao passar o mouse sobre "HTML", será exibido o texto "Hypertext Markup Language".)
